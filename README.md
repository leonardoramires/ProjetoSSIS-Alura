# ProjetoSSIS-Alura

O projeto presente neste diretório tem como objetivo construir um fluxo de dados para Business Intelligence utilizando SQL Server Integration Services. Cada um dos passos executados seguem as orientações passadas no <a href='https://cursos.alura.com.br/course/business-intelligence-sql-server-e-integration-services'>Curso de ETL com Integration Services: Modelo de dados</a> da escola de cursos online <a href='https://www.alura.com.br/'>Alura</a>.
